<h1>How to run Scribe</h1>

<p>
You can run Scribe in the following ways:
<li> <b>SaaS</b>: Scribe Hub web application (prod.hub.scribesecurity.com) provides a browser-based experience, along with functions such as configuration settings, managing evidence sharing, and reports.</li>
<li> <b>On-prem</b>: Scribe Valint Command Line Interface or plugins to CI products enable you to collect evidence and apply your security policies to it.</li>
</p>
